eSciDoc Browser 0.3.3 must be deployed in JBoss Application Server 4.3.x.

!!!!! For detailed instructions see https://www.escidoc.org/wiki/ESciDoc_Browser. !!!!!

Put the Web Archive file ''browser.war'' in 
[escidoc-directory]/jboss/server/default/deploy

If a proxy server must be used for internet access configure proxy settings for JBoss.
E.g. by giving the following Java options:
	JAVA_OPTS=-Dhttp.proxyHost=proxy.[host-name] -Dhttp.proxyPort=[proxy-port-number] -Dhttp.nonProxyHosts=localhost|127.0.0.1

After starting JBoss the eSciDoc Browser is accessible via JBoss URL and the context ''browser''.
	http://[escidoc-jboss-host-name]:[port-number]/browser?escidocurl=http://[escidoc-hostname]:[portnumber]
E.g.
	http://escidoc.fiz-karlsruhe.de:8080/browser?escidocurl=http://escidoc.fiz-karlsruhe.de:8080
