README 

The eSciDoc Browser is a Rich Internet Application that allows browsing of 
digital assets which are stored in an eSciDoc Core infrastructure. It also 
provides means to create and manipulate content resources. 

Since version 0.3.3 eSciDoc Browser includes the BW-eLab extension.

For more information on the eSciDoc Browser and getting started with the
BW-eLabs extension see "eSciDoc Browser Wiki" 
https://www.escidoc.org/wiki/ESciDoc_Browser
